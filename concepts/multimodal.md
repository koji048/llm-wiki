# Custom GPTs

**Custom GPTs** (also called GPTs) are no-code custom assistants created in ChatGPT without any programming.

## Features

- **Custom system prompt**: Define persona and behavior
- **Knowledge files**: Upload PDFs, documents for the GPT to reference
- **Actions**: Define APIs the GPT can call
- **Specific capabilities**: Enable/disable web browsing, DALL-E, etc.

## Use Cases

- Domain-specific tutors
- Code reviewers with style guidelines
- Writing assistants for specific formats
- Research assistants with document knowledge

## vs Fine-tuning

Custom GPTs customize through prompting and retrieval, not retraining. Fine-tuning modifies actual model weights and requires technical expertise.
