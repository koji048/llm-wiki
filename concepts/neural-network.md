# Reasoning Models

**Reasoning models** (like OpenAI's o1, o3) are specialized for deep reasoning and complex problem-solving.

## How They Work

- Spend more compute "thinking" before responding
- Use reinforcement learning specifically for reasoning tasks
- Internally generate long chains of thought

## Strengths

- Complex mathematics
- Competitive programming
- Multi-step logical problems
- Scientific reasoning

## When to Use

- Hard math problems
- Competitive programming challenges
- Complex debugging
- Scientific analysis

## Limitations

- Slower than standard models
- May be overkill for simple tasks
- More expensive
