# Neural Network

A **neural network** is a computational model inspired by biological neural networks, consisting of interconnected nodes (neurons) organized in layers.

## Structure

- **Input layer**: Receives raw data
- **Hidden layers**: Learn representations
- **Output layer**: Produces predictions
- **Weights**: Parameters learned during training

## Key Concepts

- **Forward pass**: Compute output from inputs
- **Loss function**: Measure prediction error
- **[[backpropagation]]**: Compute gradients
- **[[gradient-descent]]**: Update weights

## Types

- Feed-forward (MLP)
- Convolutional (CNN)
- Recurrent (RNN)
- Transformer (attention-based)
