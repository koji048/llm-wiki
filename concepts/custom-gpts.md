# Prompting

**Prompting** is the practice of crafting inputs to elicit desired outputs from language models.

## Key Strategies

| Strategy | Description |
|----------|-------------|
| **Zero-shot** | No examples, just ask |
| **Few-shot** | Provide examples of input/output |
| **Chain-of-thought** | Ask for step-by-step reasoning |
| **System prompt** | Set persona and constraints |

## Best Practices

- Be specific and clear
- Provide context when needed
- Use examples for complex tasks
- Break complex tasks into steps
- Iterate and refine prompts
